# superstore-sales-analysis

Exploratory data analysis of a US retail dataset (9,994 orders, 2014–2017).

## Questions Explored
- Which year had the highest revenue?
- Which region performs best?
- Which product category is the most profitable?
- Which month generates the most sales?

## Tools
Python · pandas · matplotlib · seaborn

## Results
| KPI | Value |
|-----|-------|
| Total Revenue | ~$2.3M |
| Best Region | West |
| Best Category | Technology |
| Best Month | November |

## How to Run
Open `Superstore-sale-Analysis.ipynb` in Google Colab and upload the dataset.
